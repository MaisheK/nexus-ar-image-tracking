# Nexus AR Image Tracking

Augmented reality application developed in Unity using Vuforia. This project enables Jawi alphabet image tracking, allowing users to point their device at printed or digital Jawi letters and view 3D models or animations linked to each character.

Features:

Tracks Jawi alphabet characters using Vuforia Image Targets.

Displays corresponding 3D models, textures, or visual effects for each letter.

Built with Unity and Vuforia for cross-platform AR support.

Optimized for mobile devices.
